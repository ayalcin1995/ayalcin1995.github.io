Course Overview

This course focuses on developing discrete event stochastic simulation models for a variety of systems including examples from manufacturing and service sectors. Special emphasis is placed on model verification and interpretation of simulation results.

Course Objectives

  1. To understand simulation modeling.
  2. To learn the art and science of collecting and analyzing data for a system under study.
  3. To learn a state-of-the-art comprehensive simulation language (ARENA) with which to develop simulation models.
  4. To use computers to debug, run and animate simulation models.
  5. To analyze and interpret results obtained from a simulation run.
  6. To provide simulation examples from a variety of systems.

Arena Software Use

This course uses Arena Simulation Software. You can download and use the free version for small models up to 300 hundred entities. The full version is available from USF Apps.
